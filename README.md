# **Jiggi**: Modern Project Management and Team Collaboration

Jiggi is a modern lightweight project management platform designed for teams and freelancers who want clarity, collaboration and real time progress tracking -- without enterprise clutter.

# Stack

- **Frontend**: React, Tanstack Query, Vite, Tailwind CSS, Zustand, gsap
- **Backend**: NodeJs and Express
- **Database** : Postgres, Sequelize
- **Others**: Socket io, Jwt
- Deployment : Render, Neon
